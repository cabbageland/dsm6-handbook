# DSM-6 Handbook

A tiny fake handbook for the *DSM-6: Extremely Contemporary Disorders Edition*.

This project is a satirical book manuscript in the style of a psychiatric handbook, covering modern afflictions like Chronic Tab Proliferation Disorder, Reply Draft Paralysis, and Chronic Ambient Administrative Dread.

## Files

- `manuscript/book.md` - main manuscript

## Current version

Version 0.1 draft includes:
- title and front matter
- preface and introduction
- all 48 disorder entries
- expanded DSM-style sections
- severity specifiers
- treatment principles
- appendices with intake questions, differential diagnosis notes, and case vignettes
